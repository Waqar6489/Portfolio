#  Sleek Tech Portfolio - Waqar Ali

A modern, fast, and fully responsive personal portfolio website built to showcase my development skills, projects, and tech stack. This portfolio features a futuristic dark-theme design with neon accents and clean filtering.

Live Demo: [https://your-portfolio-link.vercel.app]

##  Features

- **Dynamic Project Filtering:** Instantly filter projects by category (Django, Flask, WordPress, Oracle Apex) without page reloads.
- **Sleek UI/UX:** Responsive layouts, modern neon gradients, glowing background orbs, and custom hover animations.
- **Custom Branding:** Features a custom-designed `< W />` vector favicon matching the modern web-developer aesthetic.
- **Tech Stack Badge Showcase:** Clean display of micro-technologies used in each project card.

##  Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Icons:** Lucide React, React icon
- **Animations:** Custom CSS Transitions, ScrollReveal
- **Deployment:** Vercel
